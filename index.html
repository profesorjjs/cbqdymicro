Fotografía a valorar

Ítems de valoración (1 = muy bajo, 10 = muy alto)

Consulta esta rúbrica si necesitas aclarar qué significa cada indicador y cómo interpretar la escala.

Originalidad y novedad
  • 1–2: Imagen previsible o imitativa, sin rasgos personales claros.
  • 3–4: Variación leve sobre lo habitual, con algunos detalles propios.
  • 5–6: Propuesta personal reconocible; combina elementos conocidos con intención.
  • 7–8: Enfoque claramente original o inesperado para el contexto educativo.
  • 9–10: Muy innovadora; sorprende por idea, enfoque o resolución.
Expresión creativa y emocional
  • 1–2: No se aprecia intención expresiva o mensaje.
  • 3–4: Intención débil o poco definida.
  • 5–6: Mensaje o emoción reconocible.
  • 7–8: Transmite con claridad un significado personal.
  • 9–10: Muy evocadora; alto poder simbólico o emocional.
Uso innovador de técnicas digitales
  • 1–2: Uso muy básico o mecánico de recursos.
  • 3–4: Uso correcto, pero sin intención creativa clara.
  • 5–6: Uso adecuado con algunos elementos creativos.
  • 7–8: Uso intencionado para reforzar la idea (luz, encuadre, edición básica…).
  • 9–10: Uso muy creativo y eficaz de recursos disponibles.
Composición visual y técnica
  • 1–2: Problemas técnicos importantes o composición caótica.
  • 3–4: Composición simple o poco cuidada.
  • 5–6: Composición correcta y equilibrada.
  • 7–8: Buena composición, coherente con el mensaje.
  • 9–10: Muy cuidada; organización visual excelente.
Interacción y cocreación
  • 1–2: No se aprecia dimensión social/colaborativa.
  • 3–4: Interacción muy limitada o anecdótica.
  • 5–6: Contexto social claro o relación con otros elementos/personas.
  • 7–8: Uso intencionado del entorno o colaboración para crear significado.
  • 9–10: Integración muy rica de interacción, contexto o cocreación.
